# Imágenes Dummy para Desarrollo

Este directorio contiene placeholders para imágenes de productos durante el desarrollo local.

## Cómo usar:

1. Las imágenes dummy se pueden servir desde el directorio `uploads/products/` del backend
2. Para desarrollo local, puedes usar URLs de placeholder como las que están en los seeders
3. En producción, reemplaza estas URLs con las imágenes reales subidas por los usuarios

## URLs de placeholder utilizadas:

- Unsplash API para imágenes de alta calidad y relevantes para cosplay
- Todas las imágenes son de uso gratuito y optimizadas para web

## Notas:

- Las imágenes se cargan desde URLs externas para evitar archivos grandes en el repositorio
- En un entorno de producción real, considera usar un CDN para las imágenes
- Los placeholders incluyen variedad de productos cosplay para testing completo
